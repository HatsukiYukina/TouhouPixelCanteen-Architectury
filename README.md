<h1>幻雀妖</h1>
一个尝试在mc实现“东方夜雀食堂”模拟经营机制的模组<br>
这里是一个重新实现的“妖怪们的归家”，支持neoforge/fabric双加载器
<h3>项目特色</h3>
-项目将会尽量还原夜雀食堂的点单机制</br>
-项目的部分食物支持在世界上放置</br>
-项目为农夫乐事及Minecrft原版的部分食物添加了大量符合东方世界观的附加说明
<h3>项目支持情况</h3>
旧版仅支持1.20.1forge平台，新版支持1.21.1neoforge/fabric平台
